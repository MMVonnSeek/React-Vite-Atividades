## Descrição
Repositório oficial contendo atividades, exemplos e pequenos projetos desenvolvidos e aplicados por mim durante as aulas de React com Vite. Material organizado para ensino, revisão e prática dos alunos.

## Estrutura sugerida
/exemplos
/componentes
/atividades
/projetos
/desafios
/README.md


## Conteúdo
- Configuração inicial com Vite  
- Componentes funcionais e organização de pastas  
- Estados e hooks (useState, useEffect)  
- Comunicação entre componentes (props, callbacks)  
- Manipulação do DOM e eventos  
- Mini projetos e exercícios guiados

## Como executar um exemplo

# instalar dependências
npm install

# iniciar servidor de desenvolvimento
npm run dev
Objetivo pedagógico
Disponibilizar um acervo didático que reflita as práticas e exemplos adotados em sala, facilitando revisão e aprendizado prático.
